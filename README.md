### Hi there 👋

I have conducted in-depth research on DiT, VideoGen, DigitalHuman, Multimodal etc.

#### DigitalHuman & FaceSwap

* 🤖 [**Duix-Mobile**](https://github.com/duixcom/Duix-Mobile) — 端侧实时交互数字人 SDK。
* 🧑‍🎤 [**Duix-Avatar**](https://github.com/duixcom/Duix-Avatar) — 开源 AI 数字人工具，支持离线视频生成与形象、声音克隆。
* 🎭 [**ReHiFace-S**](https://github.com/ykk648/ReHiFace-S) — 实时高保真换脸算法，支持零样本推理与 ONNX 部署。
* 🎯 [**XSegNet2onnx**](https://github.com/ykk648/XSegNet2onnx) — DeepFaceLab XSegNet 权重转 ONNX 工具，推理提速至 90ms。
* 📷 [**face_power**](https://github.com/ykk648/face_power) — 人脸算法工具库，从 AI_Power 独立出的人脸专用模块。

#### Foundation Libraries

* 🧠 [**AI_power**](https://github.com/ykk648/AI_power) — AI 工具箱与预训练模型集合，覆盖 CV、音频、视频生成等多领域。
* 🔊 [**audio_power**](https://github.com/ykk648/audio_power) — 音频处理工具库，封装 TTS、ASR、音频特征提取等常用接口。
* 🧩 [**cv2box**](https://github.com/ykk648/cv2box) — 工作中高频使用的 CV 工具函数集合。
* ⏱️ [**apstone**](https://github.com/ykk648/apstone) — ONNX 模型推理高级封装，用于下游项目。
* 🧠 [**llm_power**](https://github.com/ykk648/llm_power) — LLM API 调用与视频生成实验工具集，集成豆包、通义千问等多平台接口。

#### VideoGen
